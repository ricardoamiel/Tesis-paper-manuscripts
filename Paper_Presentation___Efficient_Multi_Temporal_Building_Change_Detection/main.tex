\textbf{\color{utecblue}Drop-In Engineering Scope:}
\vskip 0.1cm
\begin{itemize}
  \item MHSA systematically replaced at \textbf{all 4} Siamese levels.
  \item Feed-Forward Networks (FFN): \textbf{Unchanged}.
  \item Inductive Skip Connections: \textbf{Preserved}.
  \item Final Segmentation Decoder: \textbf{Intact}.
\end{itemize}

\vskip 0.5cm
\textbf{\color{charcoal}Complexity Profile Per Level:}\\
\hspace{0.5em}Standard MHSA:\; $\mathcal{O}(N_l^2\,d)$\\
\hspace{0.5em}Rank-Augmented RALA:\; $\mathcal{O}(N_l\,d^2)$\\[4pt]
Given that $N_l \gg d$ holds at every hierarchical layer, it delivers a sustained \textbf{sub-quadratic memory and operational profile}.
}
\end{column}
\end{columns}

\end{frame}

% FRAME 8 — Mathematical Innovation: RALA Core
% ----------------------------------------------------------
\begin{frame}{Mathematical Innovation — Rank-Augmented Linear Attention}

\begin{columns}[T]
\begin{column}{0.42\textwidth}
\centering
\includegraphics[width=\linewidth,height=0.58\textheight,keepaspectratio]{RALA.png}\\
\vspace{2pt}
{\tiny\color{charcoal}\textit{Fan et al., ICLR 2025}}
\end{column}

\begin{column}{0.55\textwidth}
\footnotesize
\textbf{\color{utecblue}Global Query Descriptor (Context Ponderation):}
\[
Q_g = \tfrac{1}{N}\textstyle\sum_{i=1}^{N} Q_i
\]

\textbf{\color{utecblue}Differential Token Re-weighting Coefficients:}
\[
\alpha_j = N\cdot\frac{\exp\!\bigl(Q_g\,\kappa(K_j)^\top\bigr)}{\sum_{m}\exp\!\bigl(Q_g\,\kappa(K_m)^\top\bigr)}, \quad \textstyle\sum_j\alpha_j=N
\]

\textbf{\color{utecblue}Rank-Augmented Key-Value Mixing Buffer:}
\[
B = \textstyle\sum_{j=1}^{N} \alpha_j\,\kappa(K_j)^\top V_j \quad \in \mathbb{R}^{d \times d}
\]

\textbf{\color{utecblue}Hadamard Output Feature Modulation:}
\[
Y_i = \phi(X_i)\odot\bigl(\kappa(Q_i)\,B\bigr)
\]
\end{column}
\end{columns}

\vskip 0.2cm
\hrule
\vskip 0.15cm
\footnotesize
\textbf{\color{utecblue}Architectural Invariant}: The pixel-dependent transformation layer $\phi(\cdot)$ effectively breaks the low-rank capacity ceiling ($\operatorname{Rank}(Y)\not\le d$). This architectural step fully restores multi-temporal semantic expressiveness while remaining within linear time boundaries $\mathcal{O}(Nd^2)$.

\end{frame}

\begin{frame}{Qualitative Analysis — S2Looking Architectural Limits}

\vskip 0.4cm
\textbf{\color{red!80!black}Core Boundary Insight — The Parallax Limit:}
\vskip 0.3cm

\begin{itemize}
  \item \textbf{Error Pattern Invariance}: Error topologies and false positive distributions remain completely identical across choices of full-rank or linear attention layers.
  \item \textbf{Backbone-Level Constraints}: The geometric failure mode originates natively in the representation limitations of the SegFormer-B2 convolutional backbone, not in the attention module itself.
  \item \textbf{Metrological Reality}: In remote sensing tasks featuring severe oblique acquisition tracking, no change in the mathematical formulation of the attention layers can recover perspective distortions without explicit stereophotogrammetric depth alignment heads.
\end{itemize}

\vskip 0.6cm
\centering
\footnotesize\color{charcoal}
This analytical boundary confirms that RALA-CF achieves maximum efficiency without introducing custom model degradation, framing the remaining challenges as a data view geometry issue.

\end{frame}

\section{Conclusions}

% ----------------------------------------------------------
% FRAME 13 — Conclusions

\begin{frame}{General Conclusions \& Architectural Boundaries}

\begin{columns}[T]
\begin{column}{0.48\textwidth}
\textbf{\color{utecblue}Core Empirical Milestones:}
\vskip 0.15cm
\begin{itemize}
  \item Map quadratic encoder limits down to strict linear complexity bounds: $\mathcal{O}(N_l^2d) \to \mathcal{O}(N_ld^2)$.
  \item Mitigate low-rank collapse paths via Hadamard product output token modulation $\phi(\cdot)$.
  \item Retain structural accuracy parity across the comprehensive 6-dataset evaluation matrix ($\Delta = +0.14$\,pp mean F1).
  \item Achieve a 22.2\% VRAM saving, 8\% GFLOPs mitigation, and an exact uniform 50\% training batch size expansion.
\end{itemize}
\end{column}

\begin{column}{0.48\textwidth}
\textbf{\color{red!80!black}Identified Research Limitations:}
\vskip 0.15cm
\begin{itemize}
  \item \textbf{Geometric Parallax Sensitivity}: Side-looking perspective deformations (S2Looking) represent a backbone limit that attention triumphs cannot solve alone.
  \item \textbf{Annotation Label Discrepancies}: Semantic multi-class class noise (SYSU-CD) triggers false positives on non-building changes.
  \item \textbf{Absolute Hardware Thresholds}: Full patch size deployment on low-tier consumer GPUs ($\leq 8$\,GB) remains out of reach.
\end{itemize}
\end{column}
\end{columns}

\vskip 0.35cm
\centering
\footnotesize\color{charcoal}
Accuracy, efficiency profiles, and error maps validate the central thesis hypothesis: Rank-augmented linear attention provides a mathematically sound, non-heuristic path toward highly efficient multi-temporal remote sensing.

\end{frame}
